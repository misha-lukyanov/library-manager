# 📚 Менеджер книг (Library Manager)

Десктопное приложение для управления личной библиотекой книг.

## ✨ Возможности

- ✅ Добавление новых книг
- ✅ Редактирование информации о книгах
- ✅ Удаление книг
- ✅ Поиск по названию и автору
- ✅ Фильтрация по авторам
- ✅ Рейтинг книг (от 0 до 5)
- ✅ Отметка о прочитанном
- ✅ Статистика библиотеки
- ✅ Сохранение в SQLite базу данных

## 📦 Установка и запуск

### Вариант 1: Запуск из исходников

```bash
# Клонировать репозиторий
git clone https://github.com/misha-lukyanov/library-manager.git

# Перейти в папку проекта
cd library-manager

# Установить зависимости
pip install -r requirements.txt

# Запустить приложение
python main.py
